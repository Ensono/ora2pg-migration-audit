# Contributing to Ora2Pg Migration Audit Tools

## Important Notice

**This project is NOT accepting external contributions at this time.**

This software is source-available but not open source. It is provided for review and 
reference purposes only.

## Why No External Contributions?

To avoid ownership ambiguity and maintain clear intellectual property rights, we are 
not accepting pull requests, issues, or other contributions from external parties 
unless there is a signed Contributor License Agreement (CLA) in place.

## Need a License?

If you would like to use, modify, or contribute to this software, please contact us 
for a commercial license.

## Internal Contributors (Ensono Employees)

If you are an Ensono employee working on this project:

1. Follow the internal development standards
2. Create feature branches from `main`
3. Submit pull requests for code review
4. Ensure all tests pass before merging
5. Update documentation as needed

## Questions?

For questions about this project or licensing inquiries, please contact the 
project maintainers at Ensono.
